# Data Corrections Examples

Examples demonstrating correction methods for time series data including offset and bias corrections.

## Contents

- **offsetcorrection.py** — Constant offset correction
- **setto.py** — Value replacement and forced corrections

## Related Documentation

See `diive.pkgs.preprocessing.corrections` for available correction classes.

## Usage

```bash
uv run python examples/pkgs/preprocessing/corrections/offsetcorrection.py
uv run python examples/pkgs/preprocessing/corrections/setto.py
```
