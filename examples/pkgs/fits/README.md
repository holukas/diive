# Data Fitting Examples

Examples demonstrating curve fitting and regression methods for time series data.

## Contents

- **fitter.py** — Polynomial and custom function fitting

## Related Documentation

See `diive.pkgs.fits` for available fitting classes and functions.

## Usage

```bash
uv run python examples/pkgs/fits/fitter.py
```

Or run all fitting examples:

```bash
uv run python examples/run_all_examples.py
```
