# Data Corrections Examples

Apply corrections and adjustments to measurement data.

## Overview

Data quality improvements and systematic bias correction:

- **Offset Correction** — Correct systematic offsets in temperature, radiation, wind direction
- **Set Values** — Replace values with missing, thresholds, or specific values
- **RH Correction** — Humidity measurement corrections
- **Measurement Bias** — Systematic measurement corrections

## Use Cases

- Correcting instrument calibration errors
- Handling systematic measurement biases
- Data quality improvements before analysis
- Standardizing measurements across datasets
