# Time Series Analysis Examples

Spectral and frequency domain analysis of time series.

## Overview

Frequency domain and spectral analysis methods:

- **Harmonic Analysis** — Spectrogram analysis for daily and annual patterns
- **Spectral Decomposition** — Identify dominant frequencies in time series
- **Temporal Dynamics** — CO₂ phenology and ecosystem cycling patterns

## Use Cases

- Understanding periodic patterns in ecosystem data
- Identifying dominant frequencies
- Analyzing daily and seasonal cycles
- Characterizing ecosystem phenology
