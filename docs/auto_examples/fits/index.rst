# Fitting Examples

Curve fitting and function fitting examples.

## Overview

Fit mathematical functions to time series data:

- **Bin Fitting** — Fit curves to aggregated/binned data
- **Interpolation** — Function fitting for gap-filling

## Use Cases

- Characterizing environmental responses
- Curve-fitting for parameterization
- Creating smooth response functions
