# Time Series Processing

Learn how to work with time series data: resampling, frequency detection, aggregation.

## Key Modules

- `diive.core.times.TimestampSanitizer` — Validate and clean timestamps
- `diive.core.times.DetectFrequency` — Detect data resolution (e.g., 30-minute)
- `diive.core.dfun` — Statistical functions for DataFrames
