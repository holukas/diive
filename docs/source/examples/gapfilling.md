# Gap-Filling

Learn about different gap-filling algorithms for missing data.

## Key Modules

- `diive.pkgs.gapfilling.FluxMDS` — Marginal Distribution Sampling
- `diive.pkgs.gapfilling.RandomForestTS` — Random Forest gap-filling
- `diive.pkgs.gapfilling.XGBoostTS` — XGBoost gap-filling
- `diive.pkgs.gapfilling.linear_interpolation` — Simple linear interpolation
